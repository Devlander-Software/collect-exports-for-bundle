Generate barrel exports automatically

collectbyexports scans your directories and generates TypeScript/JavaScript export statements for barrel files, large projects, and design systems.

Get started View on GitHub
export { Button } from './Button'
export { Card } from './Card'
export { Modal } from './Modal'
// ... automatically kept in sync

Generates exports based on your project structure.

Before vs After

MANUAL
// src/components/index.ts — you wrote this by hand
export { Button } from './Button'
export { Card } from './Card'
export { Modal } from './Modal'
// ... 47 more lines
GENERATED
// src/components/index.ts — generated
export { Button } from './Button'
export { Card } from './Card'
export { Modal } from './Modal'
export { Tabs } from './Tabs'
export { Input } from './Input'
// ... automatically kept in sync

Install & Run

npm install collectbyexports
collectbyexports collect
Configuration
// .collect-exports.json
{
  "rootDir": "src",
  "allowedExtensions": [".ts", ".tsx"],
  "ignoredExtensions": [".test.ts", ".stories.tsx"]
}

Use it programmatically

import { autoExporter } from 'collectbyexports'

await autoExporter({
  rootDir: 'src/components',
  allowedExtensions: ['.ts', '.tsx'],
  ignoredExtensions: ['.test.ts'],
})

Optimize build scripts and tooling.

Fine-grained control

// InternalHelper.ts
/** @collect-exports-exclude */
export const debugOnly = () => { /* ... */ }

Exclude individual exports with a simple directive.

Three ways to run it

CLI

Run via CLI. Build scripts, CI/CD.

collectbyexports collect
npx collectbyexports collect

Programmatic

Use the API inside your Node scripts or custom tooling workflows.

Interactive

Guided setup for new projects.

collectbyexports init --interactive

Built for design systems & shared code

Design systems

Barrel files for components, icons, tokens.

Shared types

Single source of truth for schemas & interfaces.

Monorepos

Per-package or per-directory barrels.

Bundling

Works with Rollup, Vite, Webpack.

Everything you need

  • Recursive directory scanning
  • TypeScript Compiler API support
  • Configurable extensions
  • @collect-exports-exclude directive
  • Presets and interactive config
  • CLI + programmatic API

CLI Quick Reference

collectbyexports collect -d ./src -o index

Uses flags for directory and output targeting.
